Project Steps
Step 1: Command-Line Setup
Capture the filename from the terminal. Write logic to check the argument count and exit the program safely if the user forgets to provide a filename.
(Reference: Chapter 13.7)

Step 2: File Handling
Open the target file in read mode. Implement a check to verify the file opened successfully (not NULL), and ensure the file is closed before the program ends.
(Reference: Chapter 22.1 & 22.2)

Step 3: Reading the Bytes
Create a 16-byte buffer. Write a loop that continuously reads blocks of data from the file into this buffer until the end of the file is reached.
(Reference: Chapter 22.6 & Chapter 7.1)

Step 4: The Hex Column
Inside the read loop, track your current file offset and print it. Then, loop through your populated buffer and print each byte as a padded, two-digit hexadecimal number.
(Reference: Chapter 3.1)

Step 5: The ASCII Column
On the same line, loop through the buffer a second time. Evaluate each byte to see if it is a printable character. If it is, print the character; if not, print a dot (.). Print a newline and repeat until the file is fully read.
(Reference: Chapter 23.5)
